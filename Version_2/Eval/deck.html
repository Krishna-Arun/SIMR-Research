Causal Clinical Reasoning Benchmark

Does a literature (PubMed) tool help LLMs reason about ICU interventions?

Models: Qwen3-32B, Llama-3.1-8B  ·  Benchmarks A/B/C  ·  100 Q each  ·  with vs without PubMed

Questions generated by gpt-oss-20B · graded by gpt-oss-20B (ruthless-mentor judge) · MIMIC-IV (PHI on-node)

12 result files across 2 models

Setup

  • A — Lab-request + Dx/Rx: agent requests labs (scored), then gives diagnosis/intervention (0/0.5/1).
  • B — Post-procedure trajectory: predict each lab Rising/Falling/Stable + confidence (1/0.5/0).
  • C — Counterfactual: which of two patients produced the observed post-state (0/1) + confidence.
  • Tool condition: "with tool" = agent may call the PubMed search tool; "without tool" = no retrieval.
  • Confidence: every answer carries a self-reported confidence → calibration vs actual accuracy (ECE).
